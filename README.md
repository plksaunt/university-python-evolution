# university-python-evolution
##关于这个仓库
作为化学专业的学生，我在做的像是一种“自救”？或者说是“赋能”。但某种意义上化学·+AI的模式也给我提供了超越一般理工科的探索空间，这个仓库正好可以见证我大学四年从python基础到利用PyTorch实现化学与AI交叉的足迹，因此我称其为大学python进化史。
